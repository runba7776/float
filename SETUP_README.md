# FloatCam Personal − 設置手順(Macなし・iPhoneのみで完結)

このフォルダには、フローティングカメラ風のWebアプリ(PWA)一式が入っています。
GitHub Pagesという無料の静的サイトホスティングを使ってインターネット上に公開し、
iPhoneのSafariから「ホーム画面に追加」することで、アイコンをタップして起動できる
アプリのような形で使えるようになります。Macは不要です。

## 含まれるファイル
- index.html, style.css, app.js … アプリ本体
- manifest.json, sw.js … ホーム画面アイコン化・オフライン対応用
- icon-192.png, icon-512.png, apple-touch-icon.png … アイコン画像

## 重要: 公開範囲についての正直な注意
GitHub Pagesで公開したページは、リポジトリを「Private」に設定しても、
発行されたURL(https://ユーザー名.github.io/リポジトリ名/)自体は
そのURLを知っていれば誰でも開ける状態になります(GitHubの仕様上、無料プランでは
URL自体に閲覧制限はかけられません)。
そのため、このアプリには**起動時にパスコードを要求するロック画面**を組み込んであります。
パスコードはサーバーには送らず、入力した端末のSafari内(localStorage)にハッシュ化して
保存されるだけなので、URLを誰かに知られても、パスコードを知らない限り中身は使えません。
ただし、ブラウザの開発者ツールでJavaScriptを直接読まれた場合に絶対に破られない、という
強度の暗号ではない点は正直にお伝えしておきます(URLを他人に教えない、が一番の防御線です)。

## 手順

### 1. GitHubアカウントを作る
ブラウザ(SafariでOK)で https://github.com/signup を開き、無料アカウントを作成します。

### 2. 新しいリポジトリを作る
右上の「+」→「New repository」を選び、
- Repository name: 任意の名前(例: my-floatcam)
- Public / Private: どちらでも可。中身を他人に見られたくなければ Private を推奨
  (前述の通りURL自体は公開される点は変わりません)
で作成します。

### 3. ファイルをアップロードする
作成したリポジトリ画面で「Add file」→「Upload files」を選び、このフォルダの中身
(index.html, style.css, app.js, manifest.json, sw.js, icon-192.png, icon-512.png,
apple-touch-icon.png の8ファイル)を**まとめて選択してアップロード**し、
「Commit changes」を押します。
※ iPhoneの場合、Files(ファイル)アプリでこのzipを解凍してから、Safari上のアップロード
画面で複数選択してください。

### 4. GitHub Pagesを有効化する
リポジトリの「Settings」→ 左メニュー「Pages」を開き、
- Source: 「Deploy from a branch」
- Branch: 「main」/ フォルダは「/ (root)」
を選んで「Save」します。数分待つと、ページ上部に
`https://ユーザー名.github.io/リポジトリ名/` というURLが表示されます。

### 5. iPhoneでアプリ化する
1. SafariでそのURLを開く
2. 「写真とカメラ」「マイク」の利用許可を聞かれたら「許可」を選ぶ
3. 共有ボタン(□に↑)をタップ →「ホーム画面に追加」
4. ホーム画面にできたアイコンをタップして起動
5. 初回はパスコードを2回入力して設定 → 以後起動するたびにそのパスコードで解除

## 使い方の補足
- 中央下のシャッターボタン(白丸)で写真、赤丸ボタンで動画の録画開始/停止です
- 撮影後は共有シートが開くので「画像/ビデオを保存」を選ぶとカメラロールに保存されます
- カメラの丸い窓は指でドラッグして好きな位置に移動、右下の角をドラッグして大きさ変更できます
- ズームスライダーは機種によって「数値ズーム」または「広角/超広角レンズの切り替え」の
  どちらかとして動作します(iPhoneの世代によりSafariが超広角レンズを別カメラとして
  認識するかどうかが異なるため、実機でご確認ください)
- 上部のURL欄に住所を入れるとブラウザとして使えますが、Google検索やX(旧Twitter)など
  一部サイトは埋め込み表示(iframe)を拒否しているため開けないことがあります。その場合は
  表示されるヒントから「新しいタブで開く」をお使いください
- 右上の鍵アイコンでいつでも手動ロックできます
